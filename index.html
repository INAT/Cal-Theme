Cal Theme

Cal Theme คือ HTML Template สำหรับใช้แสดงผลชั่วคราว ในขณะที่เว็บไซต์หลักกำลังพัฒนาหรือปรับปรุงอยู่ และยังไม่พร้อมที่จะแสดงผลในขณะนั้น

คุณสมบัติ

  • พัฒนาด้วยภาษา HTML เวอร์ชั่นที่ 5
  • พัฒนาด้วยภาษา CSS เวอร์ชั่น 3 โดยใช้ CSS Preprocessor ยอดนิยมอย่าง SASS
  • พัฒนาด้วย Bootstrap Framework เวอร์ชั่น 3
  • ผ่านการตรวจสอบ The W3C Markup Validation ของ W3C โดยไม่มีข้อผิดพลาดหรือคำเตือน
  • ได้รับคะแนน 100/100 ในส่วน ประสบการณ์ของผู้ใช้ สำหรับการตรวจสอบบน GooglePageSpeed Insights
  • รองรับการแสดงผลได้หลากหลายขนาดหน้าจอ (Responsive)

สัญญาอนุญาต

ท่านสามารถนำ Cal Theme Version 2 นี้ไปใช้งานได้อย่างอิสระ บนพื้นฐานของ สัญญาอนุญาตของครีเอทีฟคอมมอนส์แบบ แสดงที่มา 4.0 International.
